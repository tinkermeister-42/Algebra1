2.3 – Combining Like Terms

When we speak, we sometimes shorten a sentence without changing its meaning. We can do the same in algebra. Expressions can be messy, so we clean them up by putting like things with like things and then adding or subtracting. This is called combining like terms.

Think of a cashier counting a mixed drawer: tens with tens, fives with fives, quarters with quarters. Once things are sorted, they’re easy to count. In algebra, we sort by the variable part and then add the coefficients.

  • I can identify like terms in an expression.
  • I can combine like terms to simplify an expression.
  • I can check my work by substitution or by reasoning about equivalence.

2.3.1 – What Are Like Terms?

Like terms have the exact same variable part (same letters with the same exponents). Only the coefficients may differ.

Examples \(2x\) and \(5x\) → ✅ like \(x^2\) and \(-3x^2\) → ✅ like \(3x\) and \(3y\) → ❌ not like \(x\) and \(x^2\) → ❌ not like \(4xy\) and \(-7yx\) → ✅ like (order of letters doesn’t matter)

The sign belongs to the term. In \(7x - 2x\), the terms are \(+7x\) and \(-2x\).

2.3.2 – How to Combine Like Terms (Sort → Group → Add)

Step 1: Sort terms by their variable part (including constants). Step 2: Group like terms together. Step 3: Add/Subtract the coefficients. Keep the variable part the same.

Example A (single variable + constants)

\[ 2x + 4 + 5x - 1 \;\to\; (2x + 5x) + (4 - 1) \;=\; 7x + 3 \]

Example B (negatives and subtraction)

\[ 7x - 2x + 6 - 3 \;\to\; (7x - 2x) + (6 - 3) \;=\; 5x + 3 \]

Example C (multiple variable types)

\[ 3x + 2y + 5 - x + y - 2 \;\to\; (3x - x) + (2y + y) + (5 - 2) \;=\; 2x + 3y + 3 \]

Simplify each:

  1. \(4x + 3x\)
  2. \(5a + 2b - 3a\)
  3. \(x + 2 + x + 5\)
  4. \(7m - 2m + 6 - 3\)

2.3.3 – Why Does This Work? (The idea behind it)

When the variable part matches, we can factor it like a common word:

\[ 3x + 2x = (3+2)x = 5x \qquad\text{and}\qquad 4y - y = (4-1)y = 3y \]

So \(3x + 2x\) and \(5x\) are equivalent expressions. They always give the same value for any \(x\).

Pick a value (e.g., \(x=10\)). Compare \(3x + 2x\) and \(5x\). What do you notice?

2.3.4 – Common Mistakes (Watch out!)

  • Different variable parts don’t combine: \(x\) and \(x^2\) are not like; \(xy\) and \(x\) are not like.
  • Don’t mix variables with constants: \(2x + 3 \neq 5x\).
  • Mind the sign: In \(8x - 5x\), the second term is \(-5x\), not \(+5x\).

If you’re unsure, test with a number (substitution). If two forms always give the same value, they’re equivalent; if not, you changed the meaning.

2.3.5 – With Parentheses: “Open, Then Combine”

If parentheses are present, simplify inside (or distribute if needed in a later lesson), then combine like terms.

Example D (reorder & combine)

\[ (4x - 3) + (2 - 5x) \;\to\; (4x - 5x) + (-3 + 2) = -x - 1 \]

Example E (a leading negative)

\[ -(3x - 4) + 2x + 7 \;\to\; -3x + 4 + 2x + 7 = (-3x + 2x) + (4+7) = -x + 11 \]

  1. \((5y + 2) - (3y - 6)\)
  2. \(-(2x + 3) + (x - 5)\)

2.3.6 – By Variable and Exponent (mixing \(x\), \(x^2\), and constants)

We can (and should) combine each kind separately:

\[ 4x^2 + 3x - 2 + x^2 - 5x + 6 = (4x^2 + x^2) + (3x - 5x) + (-2 + 6) = 5x^2 - 2x + 4 \]

  1. \(2x^2 + 5x - 3 + 4x^2 - x + 1\)
  2. \(7a^2 - 2a + 4 - 3a^2 + a - 6\)
