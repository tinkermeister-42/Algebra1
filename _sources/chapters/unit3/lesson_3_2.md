# Solving Two-Sided Equations

Today we're going to learn how to solve equations that have variables on both sides of the equals sign. This is an important step in understanding how equations work and how to keep both sides balanced as you simplify and solve.

```{admonition} 🎯 Objectives
- Solve equations with variables on both sides
- Use inverse operations to isolate variables
- Recognize when equations have no or infinite solutions
```

```{note}
**Vocab:** equation, variable, inverse operations, like terms
```

## 🔥 Warm-Up
Try the following problems. See what you remember — and don’t worry if the last one feels tricky.

1. Simplify: $3(x + 2) - 4$
2. Combine like terms: $5x - 2x + 7$
3. Challenge: $2x + 1 = x + 5$

## 📘 Let’s Learn It

We can solve equations that have variables on both sides by:

1. Simplifying each side.
2. Moving all variable terms to one side.
3. Moving constants to the other side.
4. Solving for the variable.

```{admonition} 🧠 Example
Solve: $2(x - 1) = 3x + 4$

$2x - 2 = 3x + 4$  
$-x = 6$  
$x = -6$
```
